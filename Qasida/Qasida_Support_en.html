Qasida Support

Qasida APP is an application that can be linked to a specific Bluetooth speaker. After linking to the speaker, it can control the playback content inside the speaker through BLE data transmission.

Users can set and adjust the AZAN time, as well as select different AZAN setting parameters to control the speaker's light on/off, display mode, etc.
